# 视觉参考素材说明

本目录下的设计稿为 AI 生成的高保真 UI 参考图，用于指导视觉还原（布局、层级、间距、动效、组件状态）。风格基调：暗色主题、留白充足、衬线标题字体、金色分隔线点缀。

| 文件 | 对应页面/状态 | 参考重点 |
|---|---|---|
| `reference_home_desktop.png` | 首页（`/`） | Hero 区排版、精选作品预览卡片样式 |
| `reference_work_desktop.png` | 作品集页（`/work`） | 分类筛选交互、masonry 网格布局与 hover 态 |
| `reference_series_desktop.png` | 系列详情页（`/work/:seriesId`） | 叙事式长图文排版、图文交替节奏 |
| `reference_lightbox_desktop.png` | 全局灯箱组件 | 灯箱交互态视觉规范、导航箭头与信息栏样式 |
| `reference_about_desktop.png` | 关于/联系页（`/about`、`/contact`） | 双栏布局、时间线样式、表单视觉规范 |
| `reference_work_mobile.png` | 作品集页移动端 | 桌面→移动响应式转换方式（网格→单列，overlay→底部说明） |

**说明**：以上均为风格参考，不要求逐像素还原；核心要求是布局结构、间距节奏、组件状态和整体质感的一致性。
